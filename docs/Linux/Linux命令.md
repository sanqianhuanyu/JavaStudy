### 常用命令

```
cd 
cd ..
cd /
mkdir
pwd
find  /home  -name  "*.txt"
find  /home  -iname  "*.txt"  忽略大小写     
touch
vi 进入编辑
IOA
dd删除一行
pp复制一行
/string  搜索
?string
:%s/old/new/g  全文替换
rm -rf 删除
tar -zcvf 打包压缩后的文件名 要打包压缩的文件
tar -zxvf 解压文件
chmod 777  r4 w2 x1
free -k   或者-m -g
top -d 3  查看系统健康状态  每隔3秒
useradd  zhangsan  用户添加
passwd zhangsan    用户密码修改
userdel    用户删除
```
