###Markdown 到 HTML格式、PNG格式之间转换

网上很难找到一个比较方便并且效果好的转换方法，最后我是通过 Visual Studio Code 的 Markdown PDF 插件完美解决了这个问题！

### 安装 Markdown PDF 插件

① 打开Visual Studio Code ，按快捷键 F1，选择安装扩展选项

**② 搜索 “Markdown PDF” 插件并安装 ，然后重启**

### 使用方法

打开一份 Markdown 文件 点击F1，然后输入export
